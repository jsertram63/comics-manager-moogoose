# comics-manager-moogoose

equivalent of comics-manager-squelize but with mongoose (Mongodb)

features :

    add a heroe category (spiderman-deadpool)
    add a comics books
    list of all comics
    details of a comics
    login 
    validator-field 
    send-mail to reset pwd

   
   configuration with mongodb could atlas
